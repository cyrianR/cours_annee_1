Comment est réalisée l'exigence E18 ?
-------------------------------------

Les attributs de la classe Cercle ont tous un modifieur private donc un utilisateur 
ne peut pas les changer les caractéristiques d'un cercle autrement qu'avec les méthodes 
fournies par la classe.



=====[ Explications ]===========================================================

(Facultatif) Donner ici les explications supplémentaires utiles à la
compréhension du travail rendu.

J'aurais voulu faire du chaînage de constructeurs pour éviter la redondance de code
mais le mini projet nous impose des fichiers de tests et donc une structure fixe
la classe Cercle. En effet, pour appeler un autre constructeur dans un constructeur
on utilise this() qui doit être placé en première position et donc cela empêche 
de définir les pré-conditions (avec assert) nécessaires aux tests de robustesse des
constructeurs. Sans cette contrainte liée au fichier test de robustesse, on aurait pu 
utiliser une méthode de fabrique à la place des constructeurs. (et bien sûr modifier le
fichier de test pour tester ces fabriques à la place d'appeler les constructeurs)
